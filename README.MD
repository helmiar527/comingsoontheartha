# 🪙 The Artha
> **Karena kelola cuan shouldn't be that hard.**

The Artha adalah aplikasi manajemen keuangan yang dirancang untuk membantu kamu memantau arus kas dengan tampilan yang modern, minimalis, dan *seamless*. Proyek ini fokus pada kemudahan penggunaan agar pengelolaan keuangan tidak lagi membosankan.

---

## ✨ Features
* **Coming Soon Interface:** Halaman penantian yang estetik dengan *countdown timer*.
* **Modern UI/UX:** Menggunakan desain minimalis dengan tipografi yang bersih.
* **Responsive Design:** Tampilan optimal di berbagai ukuran layar (Mobile & Desktop).
* **Social Integration:** Terhubung langsung ke kanal media sosial pengembang.

## 🛠️ Tech Stack
* **Framework:** [Bootstrap 5.3.2](https://getbootstrap.com/)
* **Icons:** [Material Design Icons](https://materialdesignicons.com/)
* **Library:** [jQuery 3.3.1](https://jquery.com/)
* **Styling:** Custom CSS (Polished Interface style)
